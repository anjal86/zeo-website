# Zeo Tourism — Homepage Override

This page follows `../MASTER.md` with the following homepage-specific decisions.

## Primary objective

Help a visitor decide one of three things within the first screen and a half:

1. Which destination or journey type fits them
2. Which package to compare
3. Whether to ask Zeo for practical guidance

## Hero

- Use one immersive destination image as the visual anchor
- Keep one primary CTA and one quiet secondary action
- Add compact trust context, not a large badge wall
- Carousel controls must never overlap the next section
- Video is optional enhancement after the image, not the initial dependency

## Visual direction

- Use soft atmospheric blue/orange light accents around section backgrounds, not neon gradients
- Prefer rounded image-led cards with clean text blocks
- Reduce editorial numbering and all-caps labels
- Keep destination photography dominant
- Use Playfair sparingly; body and interface remain modern sans-serif

## Homepage section rhythm

- Hero: immersive and concise
- Planning choices: compact, highly scannable
- Destinations: balanced image grid, not an overly irregular magazine layout
- Journey types: clear comparison with one featured pilgrimage option
- Trust: factual and easy to scan
- Testimonials: one focused story plus obvious controls
- Final CTA: one decisive action with direct support options

## Responsive targets

- 375px: one-column, no clipped controls, 16px body text, 44px targets
- 768px: two-column cards where useful
- 1024px: controlled asymmetry without visual imbalance
- 1440px: maximum content width maintained; avoid stretched text

## Motion level

Motion dial: 3/10.

Only use:

- 150–250ms hover/press feedback
- carousel crossfade/slide
- optional subtle image scale on hover
- no decorative particles, cloud layers or scroll-bound parallax

## Density level

Density dial: 4/10.

- Spacious major sections
- Compact planning and trust information
- Avoid large empty zones created solely for visual drama

## Variance level

Variance dial: 5/10.

- Modern and distinctive
- Balanced grid with limited asymmetry
- No brutalist/editorial sharpness
